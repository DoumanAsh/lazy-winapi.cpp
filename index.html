lazy-winapi.cpp

Simple and lazy wrapper over WinAPI

Requirements

  • C++ compiler;
  • Windows platform;
  • CMake (to build from source).

Usage

Build from source

Headers are together with source code in src/lazy_winapi

1 mkdir build
2 cd build/
3 cmake ../
4 cmake --build .

Options:

  • UNIT_TEST - build unit tests;
  • BUILD_SHARED - build shared library instead of static.

Modules

Each module has a corresponding .cpp & .hpp files. Module are independent of each other so you're free to take a particular module sources. And import it into your project.
